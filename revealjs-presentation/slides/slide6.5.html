Paso Resultado

Fórmulas Utilizadas

1. Muestra para población infinita:

n = (z² × p × q) / e²

n = (1.96² × 0.5 × 0.5) / 0.1²

n = 96

2. Corrección para población finita:

n' = n / (1 + (n-1)/N)

n' = 96 / (1 + 95/297)

n' = 73
